# Proyecto-integrador
Codificación de código para juego snake
El objetivo del programa es desarrollar el videojuego clásico Snake utilizando todas las herramientas en lenguaje python mostradas durante las clases, además de implementar la biblioteca Tkinter.

Este código nos permite:
- Navegar por un menú interactivo
- Jugar diferentes partidas del juego Snake
- Pausar el juego cuando se desee
- Detectar colisiones con las paredes o con el cuerpo de la serpiente
- Guardar puntajes con nombre del jugador
- Visualizar reglas y puntajes almacenados
